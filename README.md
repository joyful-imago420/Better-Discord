#  BetterDiscord

**BetterDiscord** is a popular client modification for Discord that enhances your experience with custom themes, plugins, and deep personalization options.

## Key Features
- Extensive custom theme support
- Powerful plugin system with hundreds of community plugins
- Deep customization of Discord's appearance and functionality
- Built-in theme and plugin manager
- Regular updates to maintain compatibility with Discord

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `BetterDiscord.exe` as Administrator.
4. **Click Install** to patch your Discord client.
5. Restart Discord and enjoy your customized experience.

**Note:** Close Discord completely before running the installer.
