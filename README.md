## 🛠️ Adding themes and plugins

Better-Discord functions by loading extra files into your Discord client. These files fall into two categories:

### Themes
Themes change the colors, backgrounds, and layout of your Discord window. To add a theme:
1. Open Discord settings.
2. Scroll to the Better-Discord section in the sidebar.
3. Click on the Themes tab.
4. Click Open Theme Folder.
5. Place your downloaded `.css` theme files into this folder.
6. Toggle the switch next to the theme name in your Discord settings to activate it.

### Plugins
Plugins add new features, such as improved media players or custom interface buttons. To add a plugin:
1. Open Discord settings.
2. Click on the Plugins tab.
3. Click Open Plugin Folder.
4. Place your downloaded `.js` plugin files into this folder.
5. Toggle the switch next to the plugin name to enable it.

You should only download files from trusted sources within the community to maintain your account safety. Never share your Discord token or personal credentials with any plugin or theme.

## ❓ Common troubleshooting tips

Most issues occur due to updates from Discord or improper file placement. If you face problems, consider these steps:

- Check for updates: New Discord updates might temporarily break the modification. Check the release page for a newer version of the installer and run it to perform a repair.
- Verify file format: Ensure your themes end in `.css` and your plugins end in `.js`.
- Clear the cache: Sometimes Discord holds onto old data. A full restart of the application clears these cache files.
- Reinstall: If issues persist, run the installer again and select the Repair or Uninstall option, then reinstall the software.
